# Очистка тестовых данных

## Автоматическая фильтрация

API теперь автоматически фильтрует тестовые товары и категории. Следующие товары **не будут отображаться** на сайте:

- Товары с названиями, содержащими: `tes1`, `test`, `тест`, `dummy`, `example`, `заглушка`, `sample`
- Категории с названиями или slug, содержащими эти же слова

## Ручная очистка базы данных

Если нужно полностью удалить тестовые данные из базы:

```bash
npm run clean:test
```

Или напрямую:

```bash
tsx scripts/clean-test-products.ts
```

Скрипт найдет и удалит:
- Все товары с тестовыми названиями
- Все связанные модификаторы и опции
- Записи в избранном
- Пустые тестовые категории

## После синхронизации с iikoCloud

После синхронизации меню из iikoCloud все тестовые товары из вашей локальной базы будут автоматически скрыты, а реальные товары из iikoCloud будут отображаться на сайте.

## Важно

- Фильтрация работает **только на уровне API** - тестовые данные остаются в базе, но не показываются на сайте
- Для полного удаления используйте скрипт `clean:test`
- После удаления тестовых данных синхронизируйте меню из iikoCloud через `POST /api/aiko/sync`

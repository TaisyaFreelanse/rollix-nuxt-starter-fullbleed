# Инструкция: Как получить координаты полигонов из Google My Maps

## Способ 1: Экспорт KML из Google My Maps

1. Откройте карту: https://www.google.com/maps/d/u/0/viewer?mid=1IfxTsTdE_9g3TkhGFnBOVru_GQCVDNo
2. Нажмите на меню (три точки) в левом верхнем углу карты
3. Выберите "Экспортировать на компьютер" или "Export to KML"
4. Скачайте файл KML
5. Используйте скрипт `parse-kml-zones.ts` для конвертации в формат базы данных

## Способ 2: Через Google Maps API (программно)

1. Используйте Google Maps JavaScript API для получения координат полигонов
2. Или используйте веб-инструмент для конвертации KML в GeoJSON

## Способ 3: Вручную через консоль браузера

1. Откройте карту в браузере
2. Откройте DevTools (F12)
3. Найдите объекты карты в консоли
4. Извлеките координаты из объектов полигонов

## Способ 4: Использовать онлайн-конвертеры

1. Загрузите KML файл на сайт типа https://mygeodata.cloud/converter/kml-to-geojson
2. Конвертируйте в GeoJSON
3. Скопируйте координаты из GeoJSON

## Формат координат для базы данных

Координаты должны быть в формате GeoJSON:
```json
{
  "type": "Polygon",
  "coordinates": [
    [
      [158.6503, 53.0194],  // [lng, lat] - ВАЖНО: сначала долгота, потом широта!
      [158.6510, 53.0194],
      [158.6510, 53.0200],
      [158.6503, 53.0200],
      [158.6503, 53.0194]   // Последняя точка должна совпадать с первой (замыкание)
    ]
  ]
}
```

**ВАЖНО:** В GeoJSON формат координат: `[longitude, latitude]` (сначала долгота, потом широта!)
